# Installation

Built and developed using Xcode 12.2 (Swift 5)

### How to run the app?

1. Clone this repo
2. Open shell window and navigate to project folder
3. Run `pod install`
4. Open `trashure.xcworkspace` and run the project on selected device or simulator
